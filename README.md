# Ninja Runner

A side-scrolling browser runner. Dodge zombies, slide under hangers, throw kunai.

**Play:** [GitHub Pages URL pending]

## Controls
- `Space` / `↑` — jump (hold for higher, tap for shorter)
- `↓` — slide
- `X` — throw kunai (3 max, +1 every 8s, pickups in world for instant +1)

## Stack
Vanilla HTML/CSS/JS. No build step. Open `index.html` in a browser.
